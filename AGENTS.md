# AGENTS.md

## Repository expectations

- Use Python 3.11+
- Keep code modular and production-style, not notebook-style
- Prefer small focused modules over one large file
- Add docstrings to public classes and functions
- Use type hints where reasonable
- Keep UI separate from detection logic
- Save failure screenshots to captures/
- Save detection events to logs/events.csv
- Do not add unnecessary dependencies
- Prefer simple, readable code over overengineering
